# exam-fronted

#### Clonar el repositorio de Github
```
git clone https://github.com/ryanezv/gapsi-exam-frontent.git
```
## Inicio
```
npm install
```

### Despliegue del proyecto
```
npm run serve
```

### (opcional) Compilar para produccion
```
npm run build
```
### Expuesto sobre el puerto 8080
```
http://localhost:8080
```
